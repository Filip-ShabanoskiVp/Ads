
# Ads
2

3
Web Application for ads, that is my first serious web application.
4
The purpose of this project  is to make it easier for users to post an ad and another user to buy that ad, so that all selected ads that a user has selected are transferred to that user's shopping cart,  which can be found in the user profile registered on the system. Some user can see all ads of the web site, but can only add some of them to shopping card if that user is registered In the system. 
5
The main functionalities in this application are: add new ad, purchase ad, recommend ad to friend and save ad.
6
The framework I use in making this web application is ASP.NET Core, which for me is one of the best framework for developing an good Web Application.
7
Programming language that I use is this application is c#.
8
Front-end of the application is made with the help of Razor pages.
9
For data persistence I chose to use postgreSql, which is for me one of the best 3 free and 
10
open-source relational database management system emphasizing extensibility and SQL compliance.
